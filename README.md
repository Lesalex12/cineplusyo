DESCRIPCIÓN: Nuestro proyecto, Cine Plus , se creó con el lenguaje de programación Python. El objetivo de Cine Plus es proporcionar cuatro pantallas que guían al usuario a través del sistema. Las ventanas incluidas son: Inicio de sesión , Taquilla , Concesiones y Pantalla final/Boleto .

La ventana de inicio de sesión es donde el usuario ingresa su información registrada previamente. Una vez ingresados ​​los datos correctos (y si el usuario presiona el botón de salida, la ventana se cierra automáticamente), se abre la ventana de Taquilla. Esta ventana permite al usuario ver las películas disponibles, los horarios y los precios de las entradas para adultos y menores. Después de seleccionar todo, el usuario presiona el botón "Siguiente", que abre la ventana de Concesiones. También hay un botón "Borrar", que restablece automáticamente todo lo seleccionado para que el usuario pueda volver a elegir. En la ventana de Concesiones, el usuario selecciona lo que desea pedir, como palomitas, bebidas, dulces y más. También puede ver el precio de cada artículo. Una vez que el usuario termina de seleccionar sus artículos, presiona el botón "Siguiente" de nuevo, que abre la ventana de Entradas (y si presiona el botón "Cancelar", la ventana se cierra automáticamente). En esta última ventana, el sistema muestra el costo de la entrada para la película que el usuario eligió. Una vez que el usuario paga, puede disfrutar de la película junto con sus concesiones.

Ventana de inicio de sesión de SCREENS. La ventana de inicio de sesión controla el acceso al sistema. En esta ventana, el usuario debe ingresar su nombre de usuario y contraseña, y el sistema verifica que ambos datos coincidan con los registrados previamente. Si la información es correcta, la ventana permite el acceso y muestra un mensaje de confirmación; si es incorrecta, se notifica al usuario para que pueda volver a intentarlo. Además, esta ventana gestiona todo el proceso de entrada. Tras validar la información del usuario, lo redirige automáticamente a la ventana de Taquilla, donde puede gestionar las funciones de cine, las entradas y todas las funciones operativas del sistema. También incluye un botón para salir o cancelar si el usuario no desea continuar.

Ventanilla de Taquilla La ventanilla de Taquilla gestiona todo el proceso de selección y compra de entradas dentro del sistema. Aquí, el usuario puede elegir una película de un menú desplegable y, una vez seleccionada, el sistema muestra automáticamente la sala correspondiente. El usuario también puede seleccionar la hora de la función mediante botones de opción, asegurándose de elegir la hora exacta a la que desea asistir. Además, la ventanilla contiene una sección que muestra imágenes promocionales de la película para facilitar su identificación. Esta ventana también gestiona la cantidad de entradas a comprar, dividiéndolas en entradas de adulto y entradas de menor, cada una con su propio precio. A medida que el usuario ingresa las cantidades, el sistema calcula automáticamente el subtotal. Finalmente, la ventanilla de Taquilla incluye un botón para borrar todos los datos si el usuario desea reiniciar el proceso y un botón "Siguiente" para continuar una vez completada la información.

Ventana de Concesiones: Esta ventana permite al usuario añadir productos adicionales a su compra. Muestra diferentes combos con sus precios, así como palomitas de maíz de varios tamaños y refrescos de diferentes sabores y tamaños. Cada producto tiene un control para configurar la cantidad y, una vez añadido, aparece en la lista de "Productos añadidos". Mientras el usuario selecciona los artículos, el sistema calcula automáticamente el subtotal de la concesión. También hay un botón para borrar todos los datos y otro para continuar.

Ventanilla de Entradas. La ventanilla de Entradas muestra claramente toda la información que el usuario seleccionó en las ventanas anteriores. Muestra los datos de la ventanilla de Entradas, como la película elegida, el cine asignado, la hora de la función y la cantidad de entradas para adultos y menores, junto con su subtotal. También muestra los productos seleccionados en la ventanilla de Concesiones, enumerando los artículos añadidos y su subtotal. Finalmente, la ventanilla calcula y muestra el importe total a pagar, combinando los costes de las entradas y las concesiones. TECNOLOGÍAS UTILIZADAS. Las tecnologías utilizadas en este proyecto fueron Python, Visual Studio Code y wxGlade, así como Google Chrome para descargar las imágenes utilizadas.

INSTRUCCIONES PARA EJECUTAR EL PROYECTO Para el correcto funcionamiento y ejecución del proyecto, es necesario descargar la carpeta que contiene los archivos de código e imagen. Luego de eso, abre Visual Studio Code. Una vez abierto, ubica el menú “Archivo” en la esquina superior izquierda. Haz clic en él y aparecerá una lista de opciones. Selecciona “Abrir carpeta” , lo que abrirá una ventana del explorador de archivos. Luego selecciona la carpeta del proyecto y ábrela. Luego de que cargue la carpeta del proyecto, aparecerán varios archivos en Visual Studio Code. Debes abrir el archivo llamado cineplus.py . Una vez abierto, puedes ejecutar el programa haciendo clic en el ícono triangular de “reproducir” en la parte superior. Otra forma de ejecutarlo es presionando Ctrl + F5 en tu teclado.
